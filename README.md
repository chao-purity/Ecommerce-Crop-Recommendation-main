# E-Commerce and Crop Recommendation System

A full-stack web application designed to empower Kenyan farmers by providing direct market access combined with AI-driven crop recommendations and expert farming advice.

## Project Overview

This platform addresses key challenges faced by Kenyan smallholder farmers by providing:
- **Direct e-commerce marketplace** connecting farmers with retailers nationwide
- **AI-powered crop recommendations** based on location, soil conditions, and market demand
- **Intelligent chatbot** for real-time farming advice and best practices

## Features

### E-Commerce Marketplace
- **User Registration & Authentication** - Separate portals for farmers and retailers
- **Produce Listing** - Farmers can upload crops with details and images
- **Product Catalog** - Browse and search available agricultural products
- **Shopping Cart & Orders** - Retailers can purchase directly from farmers
- **Real-time Inventory** - Dynamic product availability updates

### AI-Powered Crop Recommendations
- **Personalized Suggestions** - Machine learning model recommends optimal crops based on:
  - Geographic location
  - Soil composition analysis
  - Seasonal weather patterns
  - Current market demand trends
- **Python Backend** - Flask API serving trained ML models for accurate predictions

### Intelligent Farming Chatbot
- **Natural Language Processing** - Understands context-specific farming queries
- **Expert Knowledge Base** - Provides guidance on:
  - Irrigation techniques and water management
  - Pest identification and control methods
  - Fertilizer application and soil health
  - Crop rotation and sustainable practices

## Technical Stack

**Frontend:**
- React.js with Functional Components & Hooks
- React Router for single-page navigation
- CSS for responsive styling
- Context API for state management

**Backend:**
- Node.js with Express.js framework
- RESTful API architecture
- JWT authentication system
- MongoDB with Mongoose ODM

**Artificial Intelligence:**
- Python with scikit-learn for machine learning
- Flask API for recommendation services
- Natural Language Processing for chatbot
- Synthetic data training for crop prediction models


### Prerequisites
- Node.js (v14 or higher)
- MongoDB Atlas or local instance
- Python 3.8+
